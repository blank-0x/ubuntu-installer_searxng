Automated deployment script for SearXNG using Docker on Ubuntu systems.

## Overview

This project provides a reproducible and controlled installation method for SearXNG.

It avoids common issues found in upstream installers by using containerization and adding validation, logging, and configuration flags.

The script handles:

- Docker installation (optional)
- System validation (Ubuntu + root)
- User permission setup
- SearXNG configuration
- Container deployment via Docker Compose
- Logging and error handling

## Features

- Argument-based configuration
- Automatic SECRET_KEY generation
- Custom port support
- Optional Docker installation
- Persistent logging
- Fail-fast execution model

## Installation

### 1. Clone the repository

git clone https://github.com/yourusername/searxng-installer.git  
cd searxng-installer

### 2. Make the script executable

chmod +x install_searxng.sh

### 3. Run the installer

sudo ./install_searxng.sh

## Usage

### Default installation

sudo ./install_searxng.sh

### Custom port

sudo ./install_searxng.sh --port 9090

### Skip Docker installation

sudo ./install_searxng.sh --no-docker-install

## Post-installation

Apply Docker group permissions:

newgrp docker

Or log out and start a new session.

## Access

Default endpoint:

http://localhost:8080

http://localhost:<port>

## Requirements

- Ubuntu 20.04 / 22.04 / 24.04
- sudo privileges
- Internet access

## Rationale

The official SearXNG installer does not support all distributions and may fail in environments such as Kali Linux due to:

- Distribution-specific assumptions
- Python packaging restrictions (PEP 668)
- Missing service integrations

This project uses Docker to provide a consistent and isolated runtime environment.

## Production Considerations

For non-local deployments:

- Set a secure SECRET_KEY (auto-generated by default)
- Use a reverse proxy such as NGINX with TLS
- Restrict access via firewall or authentication
- Consider rate limiting and monitoring

## About

SearXNG is a privacy-focused metasearch engine that aggregates results from multiple sources without tracking users.
